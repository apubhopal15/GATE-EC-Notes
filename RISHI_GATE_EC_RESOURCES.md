## Daily Practice Rule

Daily paper size: 33 questions as per GATE EC style.

- General Aptitude: 5 questions
- Engineering Mathematics: 4 questions
- EC Technical: 24 questions
- Main format: MCQ
- Level: GATE EC level or slightly above

Every question paper should include this option:

`Question is wrong/incomplete`

If the student selects this option, the question must be reviewed. If it is confirmed wrong or incomplete, it should be excluded from scoring.

## Scoring Rule

Percentage = Correct Attempted x 100 / Valid Questions

Valid Questions = Total Questions - Confirmed wrong/incomplete questions

Pass mark: 60%

Below 60%: Need Improvement

## Checking And Teaching Rule

After answer submission, analysis must show:

- Question given
- Student answer
- Attempted or not attempted
- Correct or wrong
- Topic/chapter
- Mistake reason
- Correct solution method with calculation steps
- Improvement plan for weak topics

## GitHub Upload Rule

Large PDFs should stay in Google Drive and be linked from GitHub. This prevents GitHub preview errors such as `Invalid PDF` and avoids broken Git LFS pointer files.

Use only study material for students. Do not use personal files such as admit card, Aadhaar, application form, photos, or payment proof in test papers.
